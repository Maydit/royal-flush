Host a Game

Select this option if you want to host a game. This means you will be keeping track of all the actions other player do(match, fold, raise, etc) during the course of the game. Once you click this button, you will be given a unqiue code that you can give to other players for them to join your session.

Join a Game

Select this option if you want to join a game. As a player, you simply input in the two cards you were given and the host will be recording your actions. Once you click this option, you will be prompted to enter a code which the host will give you in order to join a session.
